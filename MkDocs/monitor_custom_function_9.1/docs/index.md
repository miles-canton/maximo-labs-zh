# 欢迎来到 Maximo Monitor 9.1 </br>自定义函数实验

!!! info
    本 Maximo Monitor 实验演示了如何打包、注册和应用自定义函数。

在本实验中，您将编写自定义函数并将其注册到 Maximo Monitor。

您将学习成功创建自定义函数 MultiplyByFactor 所需的步骤，该函数将输入项乘以一个因子。您将定义输入/输出项，实现该函数，使用 execute_local_test 在本地测试它，将其注册到 Maximo Monitor，并将其应用于机器人设备类型。本教程使用 PyCharm，但也可以使用其他支持 Python 3.11.x 的 IDE。自定义函数启动包单独提供。
 </br></br>


练习内容包括：

* 使用 IoT Functions 框架编写自定义函数
* 打包您的函数并将其存储在 GitHub 仓库中
* 在注册前在本地测试您的函数
* 将您的函数注册到 Maximo Monitor
* 将您的自定义函数应用于设备类型

!!! note
    完成整个实验的预计时间：1 小时
