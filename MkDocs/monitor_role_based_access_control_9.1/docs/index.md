# 欢迎来到 Maximo Monitor 9.1</br>基于角色的访问控制（RBAC）实验

!!! info
    本节介绍**RBAC（基于角色的访问控制）**，这是Maximo Monitor 9.1版本中添加的强大安全功能。

RBAC允许管理员精确控制哪些用户可以访问特定页面并在Monitor应用程序中执行某些操作。
通过将用户分配到**安全组**，RBAC确保每个用户只能看到并与与其角色相关的功能进行交互。
</br>

### RBAC的主要优势

* 通过限制对敏感配置的访问来提高安全性
* 通过减少UI混乱来增强用户体验
* 使用可重用的**安全组**轻松管理权限
* 支持现实世界的团队角色，如管理员、查看者和操作员

### 您将学到什么

在本指南中，我们将涵盖：

* 什么是安全组以及如何配置它们
* 如何创建用户并将其分配到一个或多个组
* 分配多个组时权限如何表现
* 演示不同的用户角色（只读和普通用户）
</br>
