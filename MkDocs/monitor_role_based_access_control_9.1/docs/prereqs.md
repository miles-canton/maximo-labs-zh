# 前提条件说明

以下是在Maximo Monitor 9.1中使用**RBAC（基于角色的访问控制）**的前提条件。

!!! attention
    本实验需要**Maximo Monitor 9.1或更高版本**
    以及对**Monitor设置和安全模块**的有效访问权限。

---

# 所有练习

RBAC模块中的所有练习要求您具备：

1. 配备现代浏览器的笔记本电脑或台式机
   – 推荐：**Google Chrome**或**Mozilla Firefox**

2. 访问具有以下条件的**Maximo Application Suite (MAS)**环境：
   - MAS的用户登录凭据
   - Monitor应用程序的**管理员级别访问权限**

3. 有效的IBM ID
   – 如果您没有，请在[此处](https://www.ibm.com/account/reg/signup?){target=_blank}创建
   – 步骤：
     - 点击`Login to MY IBM`
     - 然后点击`Create an IBM ID`

4. RBAC相关配置访问权限：
   - 能够访问**Suite > 安全性 > 安全组**
   - 能够访问**Suite > 安全性 > 用户**

---

!!! tip
    并非所有MAS用户都具有管理员访问权限。在继续之前，请与您的环境管理员确认。
---

# 确认环境访问

在开始之前，请验证：
- 您可以登录到MAS环境
- 您可以访问**安全性**菜单
- 您能够创建或修改安全组和用户
