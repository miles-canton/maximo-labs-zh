# 欢迎来到 Maximo Monitor 9.1 </br>设备和设备类型设置实验

!!! info
    本 Maximo Monitor 实验演示了如何设置设备和设备类型。

在本实验中，您将学习如何在 Maximo Monitor 9.1 中设置设备和设备类型。

本实验概述了创建、配置和管理设备类型的过程，包括添加指标、维度和计算指标。它还详细说明了设备创建和修改的过程，以及建立设备、资产和位置之间关系的方法。此外，它还提供了有关仪表板、归档或删除设备类型的指导...</br></br>

练习内容包括：

* 设备类型创建
* 设备类型中的指标
* 设备类型中的维度
* 添加和编辑设备
* 设备与资产和位置的关系
* 计算指标
* 仪表板
* 归档/恢复/删除设备类型
* 享受乐趣

    完成整个实验的预计时间：1 小时
