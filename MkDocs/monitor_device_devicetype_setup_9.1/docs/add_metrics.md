# 目标
在本练习中，您将学习如何在设备类型中添加/编辑指标。

---
*开始之前：*  
本练习要求您：

1. 完成[所有实验](prereqs.md)所需的前提条件
2. 完成前面的练习
 
---

## 添加指标

导航到所需设备类型中的指标选项卡，然后点击 `添加指标`。
![Add Metrics](img/add_metrics01.png)</br></br>

在弹出窗口中点击 `添加指标` 以启动指标创建过程。
![Add Metrics](img/add_metrics02.png)</br></br>

在弹出窗口中，输入指标名称、显示名称、事件名称、类型和单位。
![Add Metrics](img/add_metrics03.png)</br></br>

要添加另一个指标，再次点击 `添加指标` 并重复输入过程。
![Add Metrics](img/add_metrics04.png)</br></br>

输入第二个指标的指标名称、显示名称、事件名称、类型和单位数据。点击 `添加` 将所有指标包含在表中。
![Add Metrics](img/add_metrics05.png)</br></br>

添加后，指标数据将在表中可见。
![Add Metrics](img/add_metrics06.png)</br></br>

点击箭头图标预览数据。最后，点击 `保存` 以存储所选设备类型的指标数据。
![Add Metrics](img/add_metrics07.png)</br></br>

## 添加第二个事件

再次点击 `添加指标` 以包含其他事件。
![Add Metrics](img/add_metrics08.png)</br></br>

在弹出窗口中，将显示设备类型先前保存的指标数据。点击弹出窗口中的 `添加指标` 以在表中插入新的空白行。
![Add Metrics](img/add_metrics09.png)</br></br>

输入新指标的指标名称、显示名称、事件名称、类型和单位。输入所有必需的指标后，点击 `添加` 将它们包含在表中。
![Add Metrics](img/add_metrics10.png)</br></br>

最后，点击 `保存` 以存储设备类型的更新指标数据。
!!! note "注意"
     要将指标指定为传入事件的默认时间戳，请选中 `将此用作默认时间戳` 复选框。此选项仅适用于 TIMESTAMP 类型的指标。
![Add Metrics](img/add_metrics11.png)</br></br>

## 编辑指标

点击编辑图标以修改现有指标数据。
![Edit Metrics](img/add_metrics12.png)</br></br>

点击 `-` 删除整个事件行。
![Edit Metrics](img/add_metrics13.png)</br></br>

要删除事件中的特定指标，请点击该指标旁边的 `-` 图标。点击 `高级选项` 打开弹出窗口以配置 JSON 路径和键。
![Edit Metrics](img/add_metrics14.png)</br></br>

根据需要输入 JSON 路径和键。请参考提供的示例以获取指导（此步骤是可选的）。
![Edit Metrics](img/add_metrics15.png)</br></br>

点击 `保存` 以应用并更新设备类型的指标数据。
![Edit Metrics](img/add_metrics16.png)</br></br>

---
恭喜您已成功在设备类型中添加和修改指标。</br>
