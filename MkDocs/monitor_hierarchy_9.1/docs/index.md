# 欢迎来到 IBM Maximo Monitor 9.1</br>层次结构实验

!!! info
    本实验演示如何从 Manage 设置层次结构并在 Monitor 中同步。

您将了解 Maximo 的以下功能：

* 创建组织
* 激活组织
* 创建站点
* 创建位置、系统
* 创建资产并分配位置
* 在 Monitor 中创建设备类型和设备
* 使用 CSV 分配设备
* 层次结构视图

Monitor 现已与 Manage 集成，实现两个平台之间的无缝同步，以提高运营效率。

层次结构直接从 Manage 导入，无需手动上传 CSV 文件，简化了设置过程。

组织激活现在需要在 Manage 中输入成本中心详细信息，从一开始就确保财务一致性和问责制。

一旦在 Manage 中激活组织，它们会自动出现在 Monitor 的层次结构部分，反映其结构和关系。

所有层次结构级别（包括位置、系统、资产、站点和组织）都在 Manage 中创建和管理，集中控制和保持一致性。

设备及其类型在 Monitor 中创建，允许对物理设备进行详细跟踪和监控。

设备只能分配或取消分配给资产和位置，确保在层次结构内进行集中和结构化的部署。
