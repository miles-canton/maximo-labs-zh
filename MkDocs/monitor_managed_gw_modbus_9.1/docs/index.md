# 欢迎来到 Maximo Monitor 9.1 </br>托管网关 Modbus 实验

!!! info
    本 Maximo Monitor 实验演示如何使用设备库中的标准设备。

在本实验中，您将学习如何在 Maximo Monitor 9.1 中设置托管网关（Managed Gateway），并从设备库中添加几个 OT 设备。

您将学习成功设置托管网关所需的步骤，使用一些虚拟模拟的变频驱动器（Variable Frequency Drives，VFD），这些设备将通过 Modbus/TCP 协议经由托管网关将设备数据传输到 Maximo Monitor，使用设备连接器库中预配置的 Lenze i550 VFD 连接器。</br></br>

![架构](img/index_02.png)</br></br>


练习将涵盖：

* 设置模拟器环境
* 创建托管网关
* 向托管网关添加设备
* 验证从 Modbus/TCP 变频驱动器到 Maximo Monitor 的数据流
* 享受乐趣

!!! note
    完成整个实验的预计时间：1小时
