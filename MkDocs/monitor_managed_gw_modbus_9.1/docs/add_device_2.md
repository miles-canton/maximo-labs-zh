# 目标
在本练习中，您将学习如何向托管网关添加第二个工业设备。</br>
这次您将基于您的设备类型而不是从托管网关添加它。

---
*开始之前：*  
本练习要求您已：

1. 完成[所有实验](prereqs.md)所需的前置条件
2. 完成之前的练习
 
请找到运行 [Modbus 模拟器](setup_simulator.md)的机器的 IP 地址。
我在本地网络上的一台 IP 地址为 192.168.1.64 的机器上运行模拟器。

---

在左侧菜单的 Monitor 部分中展开设置并选择 `设备类型`：</br>
![转到设备类型](img/add_device_21.png)</br></br>

按您的姓名首字母缩写筛选，选择您的 VFD 设备类型，然后点击 `添加设备`：</br>
![添加新设备](img/add_device_22.png)</br></br>

` 使用设备库` 将自动被选中，因为此设备类型基于库中的设备。</br>
筛选以在第一个下拉列表中查找并选择您的网关类型：</br>
![选择网关](img/add_device_23.png)</br></br>

在第二个下拉列表中选择您的托管网关：</br>
![网关已选择](img/add_device_24.png)</br></br>

您现在应该看到您的网关类型和名称。点击 `继续`：</br>
![选择 Modbus TCP](img/add_device_25.png)

!!! note
    添加新设备并将其添加到托管网关作为流程的自然部分进行处理。这意味着第一个下拉列表中仅显示托管网关类型，第二个下拉框中仅显示所选托管网关类型的实例。

</br>
查找并选择 Lenze i550 设备并点击 `下一步`。</br>
选择 `Modbus TCP` 协议。</br>
使用模拟器的 IP 地址和端口 20502，如 `192.168.1.64:20502`。</br>
点击 `下一步`；</br>
![定义 IP 地址](img/add_device_26.png)</br></br>

将设备 ID 定义为 `XX_Lenze_i550_02`，其中您将 XX 替换为您的姓名首字母缩写。</br>
点击 `下一步`：</br>
![配置设备 ID](img/add_device_27.png)

!!! tip 
    设备类型已被选择，因为流程知道新设备是基于设备类型添加的。</br>

</br>
将数据频率定义为 60000（60秒）。</br>
选择所有标准指标和一个额外的合成指标，这也是为第一个设备选择的。</br>
点击 `保存`：</br>
![配置设备类型](img/add_device_28.png)</br></br>

您现在可以看到两个 VFD 设备：</br>
![查看设备](img/add_device_29.png)</br></br>

在左侧菜单的 Monitor 部分中展开设置并选择 `网关`：</br>
![导航到网关](img/add_device_30.png)</br></br>

筛选以查找并选择您的网关</br>
- 选择它，您还应该看到由您的托管网关处理的两个 VFD 设备：</br>
![查看您的托管网关](img/add_device_31.png)</br></br>

---
恭喜您已成功向托管网关添加另一个工业设备。</br>
