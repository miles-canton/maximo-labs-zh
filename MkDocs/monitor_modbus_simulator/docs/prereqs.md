# 前置条件说明
以下是 Maximo Monitor Modbus 模拟器实验所需的前置条件。

!!! attention
    本实验可与任何版本的 Maximo Application Suite 一起使用，因为它独立运行。</br>
    

# 所有练习

所有练习要求您具备：

1. 一台配备 Chrome 或 Firefox 浏览器和互联网连接的计算机。
2. 访问 Maximo Application Suite 环境的用户权限。您的练习协调员应该已经向您提供了访问信息。
3. 一个 IBM ID。如果您没有 IBM ID，可以在[这里](https://www.ibm.com/account/reg/signup?)获取：<br>
o 点击 `Login to MY IBM` 按钮<br>
o 点击 `Create an IBM ID` 链接<br>
4. 测试您对 Maximo Application Suite 环境的访问。
