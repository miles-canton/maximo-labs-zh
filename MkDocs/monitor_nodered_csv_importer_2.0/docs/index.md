# 欢迎来到</br>Maximo Monitor Node-RED CSV导入器实验
您将学习如何使用CSV文件和Node-RED通过MQTT将历史数据导入Maximo Monitor（在MAS上）。

在本实验中，您将学习成功导入历史数据所需的步骤。</br>
这些步骤将涵盖：

* 设置本地Node-RED环境
* 使用CSV文件和Node-RED直接将设备数据注入Monitor
* 使用CSV文件和Node-RED通过网关将多个设备的数据注入Monitor
* 享受乐趣

!!! note
    完成整个实验的预计时间：少于1小时
