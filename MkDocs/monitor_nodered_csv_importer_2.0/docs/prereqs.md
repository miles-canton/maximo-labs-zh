# 前提条件说明

以下是为MAS Monitor而非Maximo Asset Monitor SaaS版本设计的Maximo Monitor Node-RED CSV导入器练习的前提条件。

# 所有练习

所有练习要求您具备：

1.  一台配备Chrome或Firefox浏览器并可连接互联网的计算机。

2.  可以访问Maximo Application Suite环境的用户权限。<br>
您的练习协调员应该已经向您提供了有关访问权限的信息。

3.  IBM ID。如果您没有IBM ID，可以在[此处](https://www.ibm.com/account/reg/signup?)获取：<br>
o 点击`Login to MY IBM`按钮<br>
o 点击`Create an IBM ID`链接

4.  测试您对Maximo Application Suite环境的访问权限。
