# 前置条件说明

以下是本 Maximo 动手实验的前置条件。

!!! attention
    本实验需要 Maximo Monitor 9.1 或更高版本。</br>
    MAS 应用程序授权必须为 `Limited` 或更高级别。

# 所有练习

所有练习要求您具备：

1.  一台配备 Safari、Chrome 或 Firefox 浏览器和互联网连接的计算机。

2.  访问可以在 Monitor 中配置 Maximo 房地产与设施管理集成的 Maximo Application Suite 环境的用户权限。这需要：</br>
o 您的用户必须有权访问 Monitor</br>
o 您的用户必须具有 Maximo Application Suite 的管理员访问权限。</br>

3.  一个 IBM ID。如果您没有 IBM ID，可以在[这里](https://www.ibm.com/account/reg/signup?){target=_blank}获取：<br>
o 点击 `Login to MY IBM` 按钮<br>
o 点击 `Create an IBM ID` 链接

4.  测试您对 Maximo Application Suite 环境的访问。

# 练习 1 和 4

专门为此集成创建一个 Maximo 房地产与设施管理管理用户。确保该账户可以访问 OSLC API。

要在 MAS Monitor 中集成 Maximo 房地产与设施管理，您需要拥有 TRIRIGA 实例的 URL、用户 ID 和密码。</br>
